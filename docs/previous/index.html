正在查找旧版本或文档？

自己动手，丰衣足食！Swig是在GitHub上 paularmstrong/swig 开源的。把仓库clone下来，check out你需要的版本，build、review，and change to you linking！

Clone仓库

$ git clone git@github.com:paularmstrong/swig.git
$ cd swig

版本列表

每个Swig版本都已经打上了相应的tag。

$ git tag --list

切到你需要的版本，安装需要的依赖

$ git checkout v0.14.0
$ make

Review The Targets In Makefile To See How To Build Documentation Or Packages

目标有时候会从在版本到版本之间切换，所以查看哪些是可用的是个不错的方法。

$ make -pn | perl -F: -ane 'print "$F[0]\n" if /^\w+\s*:/' | sort
